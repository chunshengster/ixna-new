什么是 Spvrk XML News Aggregator(Spvrk)

Spvrk Xml News Aggregator(IXNA)是点燃灵感(www.Spvrk.com)的子站点,由 剑气凌人提供技术支持与日常维护它的任务是收集会员在个人站点上提供的新闻及日志,并对收集后的新闻进行整理,并发布在IXNA中我们使用在国内比较流行的"聚合"(Aggregate)一词来表示这个收集、整理、发布的过程

IXNA 是怎样工作的

Spvrk 的聚合工作是全自动客户端刷新的,当然要服务器端处理.当你打开页面后,就开始刷新了,对数据库进行更新,随后对收集来的数据进行整理和分类; 最后将它们发布在 Spvrk 网站中.这一聚集与合成的工作每15分钟进行一次,从而保证 Spvrk 能为访问者提供比较新的信息,目前主要以Design、Paint、CG、UI、Web为主

怎样加入 IXNA

内容健康,没有违法行为,长期提供技术信息的 Blog 或新闻站点,都可以在申请加入 Spvrk XML News Aggregator(IXNA)，申请加入时请注明您的站点及 XML 的 URL 地址,以及简短的描述,我们会对您的站点进行评价,并保留添加和删除贵站的权利，推荐使用feedsky和feedburner烧录站点

请注意,为保证内容质量,同时也为了避免法律问题,我们不欢迎如下站点:

  • 使用免费域名或空间,经常性更改域名,无法确定长期能使用的站点
  • 包含大量转载内容的站点
  • 人云亦云,没有主见,没有个性的站点
  • 只包含设计资源的设计类站点
  • 只包含个人感情交流的站点
  • 有侵权行为的站点,这些行为包括提供盗版信息、非法转载、严重抄袭、侵犯著作权等
  • 刚开办不久或内容少于20篇的站点
  • 无法在低带宽环境下正常浏览的站点,如果您的站点包括自动播放的视频/音频内容,我们建议您关闭它再申请加入IXNA
  • 站点访问困难,或兼容性低下(以 Firefox 1.0 和 IE 6.0 能正常访问为底限)
  • 包含大量政治言论、色情内容、大量赚钱广告的站点
  • 传播迷信内容、提供大量占卜网站链接的站点
  • RSS XML ATOM的文档不标准,输出时间或时区有问题而无法聚合的站点
什么是 RSS / RDF / ATOM

目前 Spvrk XML News Aggregator(IXNA) 支持的 XML 新闻格式有三种:RSS / RDF / ATOM 它们都是互联网上比较流行的新闻描述语言,并授到大部份新闻阅读软件的支持任何一个新闻站点,只要提供了其中一种文件,我们就能使用阅读软件 或聚合程序阅读到其中的新闻它们为方便、快捷的阅读方式提供了可能,并且使新闻能被整理、分类、重新发布

我想与 Spvrk XML News Aggregator(IXNA) 交换链接,怎么做

Spvrk 仅与同类站点及合作站点交换链接,要求被链站点具有一定的访问量,同时必须符合点燃灵感交换链接中所有要求有意者请至论坛管理区联系管理人员
