[Minenews_vote 功能说明]
    本程序基于ASP版通用投票系统完成，使用文本方式存储数据，无需MySQL支持。

    自定义前台显示flash的主色调，文字颜色以及字体大小，打造自己的风格。
    
    前台显示flash有相当大的弹性，所以对文字多少没有苛刻的限制。 
    
    前台显示flash可以通过其中的翻页来查看所有开放的投票项目。
    
    强大的后台管理功能。
    
    前台显示flash调用非常简单，只需要调用一个js文件即可。
    
    具有投票项目打开关闭的功能。
    
    防止多次提交。
    
    详细使用方法请登陆后台参见帮助

[Minenews_vote 安装方法]

 1.如果您用的是 LINUX 类型的主机，请设置以下目录和文
件的权限为 777 ; 如果您用的 WINDOWS 类型的主机，那么
请直接到第2步。

 ./news_vote/data.php

 2.配置config.php中管理员用户名及密码。

[Minenews_vote 运行环境]

 Minenews_vote 运行在支持 PHP 的 WINNT/LINUX 主机上。
 
 推荐使用 Apache 2.0.48 ++ ,PHP 4.3.4 ++ , 未开启安全
模式

[Minenews_vote 后台管理]

 登陆地址：http://你的主机地址/news_vote/admin.php

 用户名和密码就是您在config.php中填的用户名和密码。
